This repository contains two files designed to demonstrate aberrant responding using the RobustIRT package in RStudio.

The “Aberrant Simulations” file provides simulated item‑response data with intentional aberrant responding patterns, such as cheating, back random responding, and suboptimal performance. It is used to:
  - Illustrate how aberrant responders affect IRT model estimation.
  - Compare standard (MLE) and robust ability estimates.
  - Demonstrate person‑fit and aberrance‑detection procedures within the RobustIRT package.

The “MTurk Demonstration” file is designed to show how RobustIRT can be applied to online testing environments where low‑effort or inconsistent responding is common. To use this demonstration:
  - Upload the MTurk dataset into RStudio.
  - Follow the accompanying script to fit IRT models, compute robust estimates, and identify aberrant responders.
