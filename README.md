This repository contains two files designed to demonstrate aberrant responding using the RobustIRT package in RStudio.
The “Aberrant Simulations” file provides simulated item‑response data with intentional aberrant responding patterns, such as cheating, back random responding, and suboptimal performance.
The “MTurk Demonstration” file is designed to show how RobustIRT can be applied to online testing environments where low‑effort or inconsistent responding is common.
